[Контрольные вопросы к лабораторным работам модуля 1](test-questions-lab-m1.pdf) Документ содержит перечень контрольных вопросов к двум лабораторным работам первого модуля «Основы машинного обучения». Вопросы охватывают полный цикл работы с данными: от поиска открытых источников и понимания структуры наборов данных до их загрузки в BI-инструменты (Yandex DataLens), генерации синтетических выборок (Faker, Sklearn, SDV), а также глубокого предварительного анализа средствами Python (pandas, numpy, matplotlib/seaborn). Блок тем включает обработку пропусков и выбросов, кодирование категориальных признаков (OneHotEncoder), методы снижения размерности (PCA, t-SNE, UMAP) и статистический анализ распределений (асимметрия, эксцесс, таблицы сопряженности, тепловые карты). Содержание полностью синхронизировано с индикаторами компетенций ML-2, BD-1 и ориентировано на формирование практических навыков ролей Data Analyst и ML Engineer согласно компетентностно-ролевой модели ИТМО.
