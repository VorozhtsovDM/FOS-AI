# Фонд оценочных средств по дисциплине «МАШИННОЕ ОБУЧЕНИЕ»

Этот репозиторий предназначен для открытого и воспроизводимого представления фонда оценочных средств (ФОС) по дисциплине «Машинное обучение». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 1. О дисциплине

#### Цель дисциплины
Цель дисциплины - изучение студентами современных методов машинного обучения, инструментов и технологий анализа данных; формирование профессиональных компетенций студентов по КРМ ИТМО в области компетенций по машинному обучению для ролей: Data Analyst и ML Engineer.

#### Основные задачи дисциплины
-	изучение теоретических основ машинного обучения; знакомство с различными научными направлениями в этой сфере;
-	изучение современных технологий сбора данных для машинного обучения;  
-	изучение методов и технологий предварительного анализа и визуализации данных.
-	изучение методов машинного обучения с учителем: регрессия, классификация;
-	изучение методов машинного обучения без учителя: кластерный анализ;
-	формирование навыков оценки и повышения качества моделей машинного обучения;
-	формирование навыков сравнения и выбора моделей машинного обучения для практических задач;
-	формирование навыков проектирования воспроизводимых пайплайнов обработки данных и обучения моделей;
-	систематизация и закрепление практических навыков и умений по решению интеллектуальных задач с использованием инструментов машинного обучения на языке Python.

#### Структура и содержание
Дисциплина состоит из 2 модулей: 1. Основы машинного обучения; 2. Классическое машинное обучение. Обучение включает лекции, лабораторные работы, проект, самостоятельную работу и завершается зачетом.

## 3. Контрольно-измерительные материалы

- [Модуль 1](M1-module-name/README.md) — шаблон описания КИМ для отдельного модуля;
- [Модуль 2](M2-module-name/README.md) — второй модуль-заготовка;
- [Проектная работа](Project/README.md) — интегральный КИМ по нескольким результатам обучения;
- [Экзамен / зачет](Exam/README.md) — промежуточная аттестация.

Каждый КИМ должен содержать назначение, проверяемые результаты, условия выполнения, материалы задания, формат сдачи, критерии и шкалу оценивания, правила использования внешних ресурсов и генеративного ИИ.

## 4. Итоговая оценка

Правила суммирования баллов, минимальные пороги и перевод результата в принятую шкалу оценок приводятся в системе итогового оценивания. Таблица должна соответствовать локальным нормативным актам образовательной организации.

## 7. Порядок заполнения

1. Заполните РПД и сформулируйте проверяемые результаты обучения.
2. Выберите релевантные профессиональные роли и компетенции КРМ.
3. Для каждого результата выберите индикатор и требуемый уровень.
4. Заполните модель измерения.
5. Создайте КИМ и рубрики оценивания по шаблонам.
6. Согласуйте систему итогового балла.
7. Добавьте ресурсы, сведения о команде и лицензию.
