# Зачет

В ходе промежуточной аттестации в форме зачета с оценкой студенту предлагается решить тест, содержащий 50 вопросов, выбираемых случайным образом.
При этом индивидуальный пул вопросов составляется следующим образом.
В него включено по 2 вопроса закрытого и открытого типа, выбранных случайным образом, на каждый индикатор.
Кроме того, в пул дополнительно включено по 1 вопросу открытого типа, выбранных случайным образом, по индикаторам ML-2.1, ML-2.2, ML-2.3, ML-3.1, ML-3.2 и ML-3.3.
