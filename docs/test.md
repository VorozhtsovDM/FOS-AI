# ОС-5. Тест (промежуточная аттестация) (вес 25%)
## Характеристика
Тест с открытыми и закрытыми вопросами. Формирующее оценивание.
## Структура теста
Раздел|Тип вопросов|Количество|Вес
|---|---|---|---|
Теоретические основы|Закрытые (выбор ответа)|10|20%
Методы МО|Закрытые и открытые|10|30%
Оценка качества|Открытые (расчёт метрик)|5|20%
Практические задачи|Открытые (кейсы)|3|30%

Примеры тестовых заданий
Закрытые вопросы:
1.	Линейная регрессия используется для:\
	A) Предсказания непрерывных значений\
	Б) Классификации объектов\
	В) Поиска ассоциативных правил\
Правильный ответ: А

2.	Целью предобработки данных является:
o	A) Увеличение размера набора данных
o	Б) Улучшение качества исходных данных перед моделированием
o	В) Уменьшение количества используемых методов
Правильный ответ: Б

3.	С какой целью данные делят на обучающий и тестовый наборы?
o	A) Чтобы обучить сеть с ранее неизвестными данными
o	Б) Чтобы тестировать сеть с ранее неизвестными данными
o	В) Чтобы сделать тестирование быстрее
o	Г) Чтобы сделать обучение быстрее
Правильный ответ: Б

Открытые вопросы:
6.	Опишите преимущества и недостатки метода k-ближайших соседей (kNN).
Ожидаемый ответ: Преимущества: простота реализации и понимания; отсутствие предположений о распределениях классов. Недостатки: высокая вычислительная сложность при больших объёмах данных; чувствительность к шумам и выбросам; необходимость выбора оптимального значения параметра k.
7.	Каково назначение градиентного бустинга в композициях алгоритмов?
Ожидаемый ответ: Градиентный бустинг предназначен для последовательного улучшения точности предсказания путём построения ансамбля слабых моделей, каждая последующая из которых направлена на исправление ошибок предыдущих.

## Ключи правильных ответов
Ключи правильных ответов должны быть подготовлены преподавателем до проведения теста и содержать:
-	Для закрытых вопросов — буквенный код правильного ответа;
-	Для открытых вопросов — модель ответа (ключевые элементы, которые должен содержать ответ);
-	Критерии частичного оценивания для открытых вопросов.
