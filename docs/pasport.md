# Паспорт ФОС
##	Наименование дисциплины и её место в ОП
Дисциплина "Машинное обучение" объемом 3 з.е. читается в 3 семестре
##	Форма промежуточной аттестации
Зачет с оценкой
##	Выбранные ИИ-роли КРМ с обоснованием (см. meta/roles_scope.md)
Профиль ИИ-роли - Data Analyst (Аналитик данных).
##	Перечень оценочных средств
| Код | Оценочное средство | Форма проведения | Тип оценивания |
|---|---|---|---|
| ОС-1 | Лабораторные работы | Практическое задание с защитой | формирующее |
| ОС-2 | Кейс-задание (case study) | Разбор прикладного кейса, письменный отчёт | формирующее |
| ОС-3 | Peer-review заданий лабораторных работ | Взаимное рецензирование по чек-листу | формирующее |
| ОС-4 | Защита итогового проекта | Презентация и демонстрация решения | суммирующее |
| ОС-5 | Тест (промежуточная аттестация) | Тест с открытыми и закрытыми вопросами | формирующее |

##	Веса оценочных средств в итоговой оценке (сумма — 100 %);

##	Единая 10-балльная шкала с сопоставлением с 5-балльной
| Балл | Уровень | Характеристика (по КРМ) | 5-балльная шкала |
|------|---------|-------------------------|------------------|
| 10 | Аналитик-визионер | Формирует стратегию, стандартизирует практики роли, признаки научной новизны или высокой практической ценности | Отлично |
| 9	| Аналитик-руководитель проекта | Формирует техническое задание на новые аналитические проекты, обосновывает выбор методов и инструментов анализа данных	| Отлично |
| 8	| Разработчик аналитического проекта | Самостоятельно создает новые аналитические проекты, обосновывает выбор методов и инструментов анализа данных | Отлично |
| 7 | Уверенный аналитик | Самостоятельно реализует роль аналитика данных на прикладной задаче, применяет лучшие практики | Хорошо |
| 6 | Самостоятельный | Самостоятельно реализует аналитический проект по типовому паттерну | Хорошо |
| 5 | Начинающий | Реализует аналитический проект по инструкции, требует консультаций по нетиповым решениям | Удовлетворительно |
| 4 | Исполнитель | Применяет инструменты по инструкции, минимально приемлемый уровень для роли | Удовлетворительно |
| 3 |Ниже порога | Частично освоил инструменты, серьёзные пробелы | Неудовлетворительно |
| 2 | Фрагментарно | Демонстрирует лишь отдельные элементы роли | Неудовлетворительно |
| 0–1 | Не сформирован | Компетенция не подтверждена оценочным средством | Неудовлетворительно |

Дисциплина считается освоенной при итоговой оценке не ниже 4,0 баллов по десятибальной шкале.
