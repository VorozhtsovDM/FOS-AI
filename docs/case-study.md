# ОС-2. Кейс-задание (case study) (вес 10%)

## Характеристика
Разбор прикладного кейса с письменным отчётом. Формирующее оценивание.

## Рубрика оценивания кейс-задания
Критерий|Вес|8-10 баллов|6-7 баллов|4-5 баллов|0-3 балла
|---|---|---|---|---|---|
Анализ предметной области|20%|Глубокий анализ бизнес-контекста, чёткая формулировка проблемы|Анализ выполнен, но неполный|Анализ поверхностный|Анализ отсутствует
Формализация задачи|25%|Корректная постановка задачи МО, выбор метрик качества|Постановка в основном корректна|Постановка неполная или некорректная|Постановка отсутствует
Обоснование решений|25%|Все решения аргументированы, рассмотрены альтернативы|Большинство решений обоснованы|Обоснования поверхностные|Обоснования отсутствуют
Практические рекомендации|20%|Рекомендации конкретны, реалистичны, с оценкой эффекта|Рекомендации есть, но недостаточно конкретны|Рекомендации формальные|Рекомендации отсутствуют
Оформление отчёта|10%|Структурированный, профессиональный отчёт|Отчёт структурирован, есть мелкие недочёты|Отчёт неструктурированный|Отчёт отсутствует
