# ОС-3 Peer-review заданий (вес 10%)

### Характеристика
Взаимное рецензирование лабораторных работ по чек-листу. Формирующее оценивание.

### Чек-лист для рецензирования
№|Критерий проверки|Да|Частично|Нет
|---|---|---|---|---|
1|Код соответствует стандартам оформления (PEP8)|□|□|□
2|Выполнена предобработка данных (пропуски, выбросы)|□|□|□
3|Проведён разведочный анализ данных с визуализацией|□|□|□
4|Использованы корректные метрики оценки качества|□|□|□
5|Проведено сравнение нескольких моделей|□|□|□
6|Обоснован выбор лучшей модели|□|□|□
7|Выводы соответствуют полученным результатам|□|□|□
8|Работа воспроизводима (указаны версии библиотек)|□|□|□

### Рубрика оценивания качества рецензии
Критерий|8-10 баллов|6-7 баллов|4-5 баллов|0-3 балла
|---|---|---|---|---|
Полнота проверки|Проверены все пункты чек-листа, выявлены существенные замечания|Проверены основные пункты, замечания есть|Проверка поверхностная|Проверка не выполнена
Конструктивность|Замечания конкретны, предложены пути улучшения|Замечания конкретны, но нет предложений|Замечания общие, неконструктивные|Замечания отсутствуют
Объективность|Оценка объективна, аргументирована|Оценка в основном объективна|Оценка субъективна|Оценка необоснованна
